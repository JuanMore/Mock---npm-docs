Deploy in seconds

After you set up an npm account, the next step is to use the command line interface (CLI) to install npm. We look forward to seeing what you create! If you choose to share your packages publicly, there is no cost.

To use and share private packages, you need to upgrade your account. To share with others, create organizations, called npm organizations, and invite others to work with you, privately or publicly.

To learn more about npm as a product, upcoming new features, and interesting uses of npm, sign up for our newsletter at npm-weekly, and be sure to follow @npmjs on Twitter.
